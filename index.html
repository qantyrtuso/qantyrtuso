🦖 О проекте

DinoNeon: Энергия будущего — динамичный 2D раннер, в котором игрок управляет кибер-динозавром. Цель — собрать как можно больше энерго-костей и установить рекорд, избегая дронов-охранников и лазерных ловушек.

3 уровня сложности: с каждым уровнем скорость бега растёт, а препятствий становится больше.

🎯 Для кого игра?

  • Школьники, которые любят динамичные аркады
  • Фанаты динозавров и киберпанка
  • Все, кто хочет провести 5 минут с пользой для реакции

📚 Польза

  • Тренирует быстроту реакции
  • Развивает внимание и концентрацию
  • Бесплатно и без интернета (после загрузки)

🛠️ Технологии

  • Игра: JavaScript + Canvas
  • 3D-модели: TinkerCAD, Blender
  • Сайт: HTML, CSS
  • Спрайты: Photoshop / Photopea
